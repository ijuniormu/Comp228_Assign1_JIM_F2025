# Comp228_Assign1_JIM_F2025
Java program with grade calculator, unit converters, and currency converter (Fall 2025).

The program includes:
1. **Grade Calculator** – Converts percentage scores into  grades.  
2. **Miles to Kilometers Converter** – Converts distance from miles to kilometers.  
3. **Liters to Gallons Converter** – Converts volume from liters to gallons.  
4. **Currency Converter** – Converts CAD to USD using a fixed exchange rate.  
5. **Driver Program** – Main class that brings everything together.

Expected output

Enter your percentage: 78
Your grade is: C
Enter miles: 42
42.0 miles = 67.578 km
Enter liters: 7
7.0 liters = 1.848 gallons
Enter CAD amount: 21
21.0 CAD = 15.33 USD

Process finished with exit code 0
